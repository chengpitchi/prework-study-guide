HTML

  • The head element contains information about the webpage.
  • The body element represents the visable content shown to the user.
  • The meta element defines metadata about an HTML docuemnt.
  • The h1, h2, ... h6 elements show different header of the content.
  • The ul element shows an unordered list on the webpage
  • The ol element shows an ordered list on the webpage
  • The li element shows the listed item witin a ordered/unordered list on the webpage.
  • The img element shows an image on the webpage with src tag containing the URL of the image.

CSS

  • A margin indicates how much space we want around the outsides of an element.
  • A padding indicates how much space we want around the content ihnsdie an element.
  • The 3 ways to style a webpage using CSS: inline CSS, internal CSS style sheet and external CSS style sheet.
  • For external CSS style sheet, a link tag need to be added to the html page for the CCS style to take effect.
  • We can use the class selector to apply CSS to specific content in our HTML page.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new brance and switches to it
  • git pull origin main: put the main branch from Github to local repository
  • git init: create a new local repository
  • git config --global user.name "User Name": Setting Git username for every repository
  • git config --global user.email "email address": Setting email address for every repository
